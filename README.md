# Spam Email Classifier

A machine learning project built using Python and Tkinter that classifies messages as Spam or Not Spam.

## Features
- Spam detection using Machine Learning
- GUI interface using Tkinter
- TF-IDF Vectorizer
- Naive Bayes Classifier

## Technologies Used
- Python
- Scikit-learn
- Tkinter
- Pandas

## Run the Project

Install libraries:

```bash
pip install pandas scikit-learn
```

Run:

```bash
python spam_classifier.py
```
